■はじめに■
・この度はPriLogをダウンロード頂きありがとうございます。

・このソフトウェアは動画から
タイムラインの自動解析ができます。

・解析可能な動画は以下の二種類です。
　PC内に保存されている動画(16:9)
　YouTubeの動画(16:9)


--------------------------------------------------------
■動作環境■
・OS Microsoft Windows 10 
・ディスプレイ 解像度1024×768以上

・解析動画 解像度のアスペクト比が16:9の.mp4動画 (推奨解像度1280x720以上)


--------------------------------------------------------
■起動方法■
・フォルダ内にある"PriLog.exe"を実行してください。


--------------------------------------------------------
■使用方法■
１、画面中央の入力フォームに解析したい動画のパス/URLを入力します。
２、「▶」を押します。
３、解析が開始されます。
４、解析完了後、結果が「result」フォルダ内に作成されます。


--------------------------------------------------------
■アンインストール方法■
・フォルダごと手動で削除してください。
